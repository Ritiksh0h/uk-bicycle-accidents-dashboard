# UK Bicycle Accidents Dashboard (1979-2018)

Interactive data visualization dashboard analyzing 40 years of bicycle accident data in Great Britain.

## Key Insights
- 25% reduction in total accidents from 1979 to 2018
- Severity increasing in recent years (15% â†’ 21% serious injuries)
- Older cyclists (66-75) face 30% serious/fatal injury rate
- Rush hour concentration: 50% of accidents during 7-9 AM and 4-6 PM

## Tech Stack
- Python
- Streamlit
- Plotly
- Pandas

## Data Source
UK Department for Transport (1979-2018)

## How to Run
```bash
pip install -r requirements.txt
streamlit run dashboard.py
```
